1) Install AAAD from github.com/shmykelsa/AAAD
2) Install TorquePro from play.google.com/store/apps/details?id=org.prowl.torque
3) Download the PerformaneMonitor app from AAAD and when asked whether to install it or not replace the pm0.apk file at .\internal\Android\data\sksa.aa.customapps\files\AAAD with the one uploaded here
4) Continue the installation process

If the installation process worked correctly, PerformanceMonitor should be available in the "Customsie launcher" section of your AndroidAuto app.

Setup TorquePro and import the custom oil pids from the OilTempPID.csv file (Settings - Manage extra PIDs/Sensors - Import CSV file).
